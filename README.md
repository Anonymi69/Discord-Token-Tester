# Discord-Token-Tester

A simple **Discord Token Tester** made in Python to test your bot tokens before using them in your application.

---

## 🚀 Installation & Usage

### STEP 1:

```bash
git clone https://github.com/Anonymi69/Discord-Token-Tester.git
cd Discord-Token-Tester
pip install -r requirements.txt
python main.py
```
